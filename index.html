DISCORD BOT SCRIPT

Run this script with Node.js on any machine (your PC, VPS, Raspberry Pi) to post Roblox deployment alerts to your Discord channel 24/7. No external packages needed — uses Node's built-in fetch.

Your Webhook URL

Step 2 — Setup

1. Install Node.js (v18+) from nodejs.org
2. Copy the script below → save as tracker.js
3. Open terminal in that folder → run: node tracker.js

# Install PM2 (process manager)

pm2 start tracker.js --name rbx-tracker

pm2 save
